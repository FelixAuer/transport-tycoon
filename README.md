# Transport Tycoon

## How to run
Example Season 1 Part 1:
```
$ php tt-1.1/transport-tycoon.php [Cargo (optional), e.g. AAABA]
```
If no input is provided `AABABBAB` is used as a default.
### Output
```
$ php transport-tycoon.php AAABA
Cargo: AAABA
Finished after 29 hours!
```
